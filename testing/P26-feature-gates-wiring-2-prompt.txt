## P26 — Câblage des feature gates admin → UI (suite de P25)

### Contexte
Suite du P25. Les 7 gates précédentes sont posées. Ce prompt câble les 6 features
restantes les plus visibles. Même pattern : `useFeatureAccess` ou `FeatureGate`.

### 1 — `weekly_checkin` — StudentWeek.tsx

**Fichier :** `src/pages/student/StudentWeek.tsx`

**Import à ajouter** (si pas déjà présent) :
```tsx
import { useFeatureAccess } from "@/providers/PlanProvider";
```

**Dans le composant**, ajouter :
```tsx
const { isEnabled: canCheckin } = useFeatureAccess("weekly_checkin");
```

**Modifier le CheckinBadge** (ligne ~926-928) pour désactiver le clic si la feature
est désactivée :
```tsx
<CheckinBadge
  checkin={currentCheckin}
  onClick={canCheckin ? () => setCheckinFormOpen(true) : undefined}
/>
```
(Quand `onClick` est `undefined`, le badge s'affiche en lecture seule — l'utilisateur
voit son check-in existant mais ne peut plus en créer un nouveau.)

**Garder** le `WeeklyCheckinForm` dans le JSX (il ne s'ouvrira simplement jamais si
`canCheckin` est false).

### Note sur `ai_week_analysis`

La clé `ai_week_analysis` dans l'admin correspond vraisemblablement à la page de bilan
IA (StudentBilan) au même titre que `ai_cycle_report` déjà gatée en P25. Vérifier
dans l'admin si ces deux keys devraient pointer vers la même feature ou des écrans
distincts — si le code n'a pas de composant séparé pour `ai_week_analysis`, cette clé
peut rester non câblée côté UI (elle sera configurable en admin mais sans effet immédiat).

### Note sur `program_creation`

La création d'un programme se fait via le bouton "Nouveau programme" dans
`src/pages/coach/StudentDetail.tsx` (section actions en bas de la fiche athlète) et
via `ProgramEditor`. Si tu veux gater la création côté coach :

Dans `src/pages/coach/StudentDetail.tsx`, chercher le bouton qui navigue vers
`/coach/students/${studentId}/program/new` (section AI Bilan + Report buttons, lignes
~470-480), et le wrapper avec :
```tsx
<FeatureGate feature="program_creation">
  <Button variant="outline" onClick={() => navigate(`/coach/students/${studentId}/program/new`)}>
    ...
  </Button>
</FeatureGate>
```
